Developed a dynamic job portal to make job application process easy using Django framework.

## Functionalities
* Create user: Job seeker or Recruiter

#### Recruiter
* Create Company
* Post Job openings with industry, salary
* Update exisiting jobs
* See details on particular jobs like: Resumes, number of applicants, etc.

#### Applicant
* Upload and edit resume
* Search jobs
* Filter jobs based on certain keywords and criterias
* Access applied jobs


## Future Scope
* Add Pagination
* Notify for particular job keywords
* Beautify the website more
